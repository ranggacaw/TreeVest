## 🔴 Critical Issues

### 1. Potential SQL Injection in Direct Query (Investment.php:216-219)
**File:** `app/Models/User.php:216-219`
**Severity:** 🔴 Critical
**Category:** `injection`

```php
public function allInvestmentTransfers()
{
    return \App\Models\InvestmentTransfer::where(function ($q) {
        $q->where('from_user_id', $this->id)->orWhere('to_user_id', $this->id);
    });
}
```

**Issue:** Direct use of `$this->id` in query builder without explicit parameter binding.  
**Recommendation:** While Eloquent typically handles this safely, consider using explicit parameter binding for critical financial queries:

```php
public function allInvestmentTransfers()
{
    $userId = $this->id;
    return \App\Models\InvestmentTransfer::where(function ($q) use ($userId) {
        $q->where('from_user_id', '=', $userId)->orWhere('to_user_id', '=', $userId);
    });
}
```

### 6. Missing Input Sanitization in Currency Formatting
**File:** `resources/js/Pages/Portfolio/Dashboard.tsx:30-32`
**Severity:** 🟠 Warning  
**Category:** `missing-validation`

```typescript
function formatCurrency(cents: number): string {
    return 'Rp ' + (cents / 100).toLocaleString('id-ID', { minimumFractionDigits: 2, maximumFractionDigits: 2 });
}
```

**Issue:** No validation for negative numbers or extreme values.  
**Recommendation:** Add input validation:

```typescript
function formatCurrency(cents: number): string {
    if (typeof cents !== 'number' || !isFinite(cents)) {
        return 'Rp 0.00';
    }
    const amount = Math.abs(cents) / 100;
    const prefix = cents < 0 ? '-Rp ' : 'Rp ';
    return prefix + amount.toLocaleString('id-ID', { 
        minimumFractionDigits: 2, 
        maximumFractionDigits: 2 
    });
}
```

{
    public const CENTS_TO_CURRENCY = 100;
    
    public static function formatCents(int $cents, string $currency = 'IDR'): string
    {
        $amount = $cents / self::CENTS_TO_CURRENCY;
        return match($currency) {
            'IDR' => 'Rp ' . number_format($amount, 2),
            'USD' => '$' . number_format($amount, 2),
            default => $currency . ' ' . number_format($amount, 2),
        };
