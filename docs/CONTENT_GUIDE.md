# Content Management Guide

## Overview

The Information & Education Center allows administrators to create and manage educational content about fruit tree investing. This guide covers the content management workflow for content authors and administrators.

## Getting Started

### Accessing the Admin Panel

1. Log in to your account with admin credentials
2. Navigate to `/admin/dashboard`
3. Click on "Articles" in the navigation or use the quick action buttons

### Creating a New Article

1. Click "Create New Article" on the dashboard navigate or to `/admin/articles/create`
2. Fill in the article fields:
   - **Title**: The main headline of your article
   - **Slug**: URL-friendly identifier (auto-generated from title)
   - **Content**: The main article body using the rich text editor
   - **Excerpt**: A short summary (shown in article previews)
   - **Featured Image**: An optional cover image
   - **Categories**: Select one or more categories
   - **Tags**: Add relevant tags for better discoverability

3. Click "Save as Draft" to save without publishing, or "Publish" to make it live

### Using the Rich Text Editor

The rich text editor supports:

- **Formatting**: Bold, Italic, Underline, Strikethrough
- **Headings**: H1, H2, H3 for structuring content
- **Lists**: Ordered and unordered lists
- **Blockquotes**: For highlighting quotes
- **Links**: Add hyperlinks to external resources
- **Images**: Upload and embed images

#### Keyboard Shortcuts

- `Ctrl/Cmd + B`: Bold
- `Ctrl/Cmd + I`: Italic
- `Ctrl/Cmd + U`: Underline

### Preview Mode

Use the preview button (eye icon) in the editor toolbar to see how your article will look when published. This helps catch formatting issues before going live.

## Publishing Workflow

### Draft Mode

Articles saved as drafts are:
- Visible only in the admin panel
- Not indexed in search
- Not accessible to public visitors

### Published Articles

Published articles are:
- Visible to all visitors
- Indexed in search results
- Eligible for related articles suggestions

### Unpublishing

To unpublish an article:
1. Go to the article edit page
2. Click the "Unpublish" button
3. The article will be reverted to draft status

## Best Practices

### Writing Educational Content

1. **Start with a clear objective**: Define what the reader should learn
2. **Use simple language**: Avoid jargon where possible
3. **Include examples**: Real-world scenarios help understanding
4. **Add visual elements**: Images and charts enhance comprehension
5. **Keep paragraphs short**: Improves readability on mobile devices

### SEO Guidelines

- Write unique meta titles (under 60 characters)
- Create meta descriptions (under 160 characters)
- Use relevant keywords naturally in content
- Add appropriate categories and tags

### Content Categories

Use these categories appropriately:

- **Getting Started**: For beginner-level content
- **Risk Management**: For content about understanding and managing risks
- **Harvest Cycles**: For explaining agricultural cycles
- **Market Analysis**: For trend information
- **Investment Strategies**: For advanced investment concepts

## Legal Disclaimers

All educational content must include appropriate disclaimers:

```
Disclaimer: This content is for educational purposes only and should not be 
considered financial advice. Past performance does not guarantee future results. 
Always consult with a qualified financial advisor before making investment decisions.
```

## Managing Media

### Uploading Images

1. In the rich text editor, click the image button
2. Select an image file (JPEG, PNG, GIF, WebP)
3. Images are automatically optimized for web display

### Image Guidelines

- Use high-quality images (minimum 800x600 pixels)
- Keep file sizes under 2MB for faster loading
- Use descriptive filenames for SEO

## Content Review

### Stale Content Alerts

The admin dashboard highlights articles that haven't been updated in over 6 months. Review these periodically to ensure accuracy.

### Analytics

Monitor article performance in the dashboard:
- View counts indicate popular topics
- Low view counts may suggest need for better SEO or promotion

## Troubleshooting

### Common Issues

**Article not appearing in search**:
- Verify the article is published
- Check that categories/tags are properly assigned

**Image upload fails**:
- Check file size (max 5MB)
- Verify file type is supported

**Slug already exists**:
- Edit the slug to make it unique
- The system will suggest alternatives
